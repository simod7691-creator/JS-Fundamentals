# JS-Fundamentals

Basic JavaScript scripts for learning the fundamentals.

## File: 0-javascript_is_amazing.js
Prints the text: *“JavaScript is amazing”* using `const` and `console.log`.

## Requirements
- Ubuntu 20.04  
- Node.js  
- First line: `#!/usr/bin/node`  
- Use `const` (no `var`)  

## Run
node 0-javascript_is_amazing.js
